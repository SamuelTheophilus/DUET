# OpenRouter Documentation.
This documentation is to provide information about the available models on OpenRouter, along with their respective rate-limits and cost-per-token.
As mentioned in the Cookbook, the budget dedicated to OpenRouter is $80. Each team member is allowed $20.
However, some team members' OpenRouter budgets may increase as unused funds are reallocated from other members.


## Models Overview.
There are over [400+](https://openrouter.ai/) models available on OpenRouter. These include both open & closed models.
Nonetheless, there are only a handful models that are recommended for programming based on benchmarks from [SWE bench leaderboard](https://www.swebench.com/)

| Model Name | Input Cost (per 1M tokens) | Output Cost (per 1M tokens) | Rate Limit |
| :--- | :---: | :---: | :---: |
| **Claude 4.5 Opus** (High Reasoning) | $5.00 | $25.00 | 43.9 billion tokens / week  | 
| **Gemini 3 Flash** (High Reasoning) | $0.50 | $3.00 | 1.4 trillion tokens / week  | 
| **MiniMax M2.5** (High Reasoning) | $0.15 | $1.15 | 229 billion tokens / week | 
| **GPT-5-2 Codex** | $1.75 | $14.00 | 5.32 billion tokens / week |
